# Employee Attendance & Payroll Backend API

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features
- Authentication & Role-based Authorization
- Employee CRUD
- Attendance (Punch In / Punch Out)
- Leave Management (Sick, Casual, Unpaid)
- Payroll Calculation with Deductions
- Pagination & Secure APIs

## Setup
```bash
npm install
npm run dev
